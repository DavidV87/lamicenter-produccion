# Lamicenter Producción

Sistema de información para control de producción de Lamicenter S.A.S.

## Tecnologías principales

* NestJS
* TypeScript
* Prisma ORM
* PostgreSQL 18
* React
* Vite

## Objetivo

Centralizar y controlar:

* pedidos
* producción
* despacho
* abastecimiento
* trazabilidad
* auditoría
* PQRS
* notificaciones

## Estado actual

Proyecto en fase inicial de arquitectura y diseño de base de datos.
